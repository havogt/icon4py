ICON Grid Domain Zone Ranges

Index-space layout of horizontal grid zones for Cells, Edges, and Vertices

Notes

  • Global grid: No lateral boundary or nudging zones. All grid points are interior. Based on R02B04_GLOBAL (20480 cells, 30720 edges, 10242 vertices).
  • Regional grid (LAM): Limited Area Model with lateral boundary rows, nudging zone, and interior. Based on MCH_OPR_R04B07_DOMAIN01 (10700 cells, 16209 edges, 5510 vertices).
  • Single rank: No domain decomposition. HALO zones are empty (start = end = total size). LOCAL spans the entire index range [0, END).
  • Multi rank: Domain decomposed across MPI ranks. Each rank owns a subset of grid points plus halo regions for communication. LOCAL spans [0, END), then HALO and HALO_LEVEL_2 extend beyond.
  • Edges have more lateral boundary levels (up to 8) than cells/vertices (up to 4) due to the staggered grid structure.
  • LOCAL = [0, END) always. In single-rank, END = total grid size. In multi-rank, END = owned count on this rank.
  • Zone ordering in index space: LATERAL_BOUNDARY → NUDGING → INTERIOR → END → HALO → HALO_LEVEL_2.
